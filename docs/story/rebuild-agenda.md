# Полное переобсуждение истории — повестка и протокол решений

**Статус:** активный рабочий процесс (открыт 21.07.2026).
**Принцип:** прежний канон — материал и аргумент, не норма. Каждое решение принимает автор. Формат блока: досье «что есть → варианты → рекомендация → решение автора».
**Правило:** решения дописываются в этот файл по мере принятия; после блока 8 они сводятся в новую редакцию канона. Проза не пишется до конца процесса.
