# N04 «Человек в промежутке» — краткая посценовая карта

**Период:** дни 12–13. **Объём:** 7 000–10 000 слов. **POV:** Александр.

1. **Перед консультацией:** Мираэль готовит собственные вопросы и идёт наверх одна.
2. **Запись перехода:** память возвращает решение войти в тело; верхняя запись независимо подтверждает время до толчка.
3. **Разрешённые страницы:** Мираэль сама определяет абзацы для земного дела.
4. **Открытый абзац:** технический сбой показывает закрытую деталь; Александр может сохранить её или немедленно закрыть. Лена прямо спорит с ним.
5. **Новая роль:** сохранение памяти, работа с материализованными хранителями, отказ от исключительной опеки и проект дисциплинарной меры.
6. **Земное показание:** три факта Мираэль связываются с геометрией машины без крови и телесных образцов.
7. **Одна классификация:** Верх получает требование выбрать служебную или физическую категорию поступка.
8. **Собственный возврат:** Мираэль принимает принцип возвращения, оставляя окончательную подпись сверке; N05 получает две ведомственные версии.

**Выбор:** сохранить случайно открытый закрытый абзац либо закрыть его без копии. Раннее сохранение можно признать до подачи, но оно повреждает автономию Мираэль и доверие Лены.

**Блокеры:** Мираэль ведёт собственное дело; Лена не говорит вместо неё; память не становится бесплатной уликой; Наблюдатель не появляется; романтика с Леной движется через конфликт доверия.
