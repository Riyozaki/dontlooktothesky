# Состояние проекта

## Проект

**Название:** Don’t Look to the Sky
**Формат:** бесплатная визуальная новелла на Ren’Py
**Язык художественного текста:** русский
**Главный POV:** Александр, близкая третья фокализация
**Общий регистр:** разговорно-пристрастный, бытовое рядом с опасным, без автоматического повышения литературного тона.

## Общая структура

```text
C00–C09 — общая ветка
M01–M07 — маршрут Мираэль
E01/E02 — концовки Мираэль
V01–V08/E03/E04 — будущий маршрут Валерии
N01–N07/E05 — будущий нейтральный маршрут
T00–T09/E06 — будущий истинный маршрут
```

## Что завершено

- Общая ветка C00–C09 написана и технически подключена.
- Маршрут Мираэль M01–M07 написан.
- E01/E02 подключены к накопленной логике прохождения.
- Для общей ветки и маршрута Мираэль проведены читательские, голосовые и рекурсивные проходы.
- V01 был написан как неудачный эксперимент, затем полностью удалён. Новая версия V01 ещё не написана.

## Текущий этап

**Активная задача:** спроектировать и написать маршрут Валерии с нуля.

Сначала требуется:

1. обсудить узлы сюжета;
2. зафиксировать карту V01 без художественной прозы;
3. сделать короткие голосовые тесты;
4. проверить их по фильтрам;
5. только после одобрения писать первую сцену;
6. расширять главу до объёма линии Мираэль, без воды.

## Технический статус

- Рабочая ветка: `arena/019f682c-dontlooktothesky`.
- Merge в `main` не выполняется без явной команды автора.
- Ren’Py SDK 8.5.3 был получен из разделённых томов, загруженных в `main`.
- Реальные `lint` и `compile` проходили через SDK.
- SDK не хранится в репозитории.
- Runtime с полноценным GUI ещё не проверен из-за headless-среды.

## Организационная структура

Организационные документы проекта (после рефакторинга 16 июля 2026) разложены так:

```text
корень/
├── SKILL.md                  — канон авторского голоса (author-voice-v2)
├── anti-cliche-filter.md     — канон фильтра ИИ-клише
├── examples.md               — реальные правки «было → стало»
└── README.md                 — единственный README, ссылки на всё

docs/
├── bible/                    — голос, персонажи, канон «кто такие»
├── filters/                  — фильтры и рекурсивные проверки
├── production/               — методика и протокол производства
├── state/                    — текущее состояние проекта
├── story/                    — сюжет: структура, маршруты, beat-sheets
├── manuscript/               — канонические рукописи и ревью (генерируются)
├── reference/                — внешние референсы
├── licenses/                 — лицензии ассетов
└── archive/                  — устаревшие и дублирующие документы
```

**Что считать источником правды для художественной работы:**

1. `SKILL.md`, `anti-cliche-filter.md`, `examples.md` в корне.
2. `docs/bible/character-bible.md` и `docs/bible/voice-and-characters.md`.
3. `docs/filters/recursive-check.md` — порядок рекурсивных проходов.
4. `docs/production/writing-protocol.md` — обязательный протокол перед написанием.
5. `docs/story/story-structure.md` и `docs/story/story-map.md` — макроструктура.
6. `docs/story/routes/*.md` — утверждённые концепции маршрутов.
7. `game/*.rpy` — единственный источник художественного текста.
8. `docs/manuscript/*.md` — автогенерируемое зеркало, **не редактируется вручную**.

## Правило объёма

Плановый ориентир маршрута Валерии — не менее масштаба линии Мираэль. Не добивать число слов механически. Объём должен появляться из:

- действий;
- погонь и последствий;
- взаимных решений;
- бытовой жизни после опасности;
- изменения отношений;
- конкретных цен и ограничений;
- самостоятельной линии Мираэль.
