# Справочный архив проекта

Эта папка обозначает материалы, которые не нужно читать перед каждой новой сценой.

Полные канонические рукописи находятся в `docs/manuscript/`. Они открываются адресно для читательского прохода, выплаты факта или проверки непрерывности.

Справочные документы (после рефакторинга 16 июля 2026):

- `../../SKILL.md` — канон голосового контракта;
- `../../anti-cliche-filter.md` — расширенный анти-ИИ-фильтр;
- `../../examples.md` — примеры авторского голоса;
- `../bible/character-bible.md` — полная библия персонажей;
- `../bible/voice-and-characters.md` — голос и речевые профили;
- `../story/common-route-outline.md` и `../story/story-structure.md` — подробная архитектура;
- `../production/production-method.md` — производственная методика;
- `../production/writing-protocol.md` — протокол перед написанием;
- `../filters/recursive-check.md` — фильтры и порядок рекурсивных проходов;
- аудиты в `../manuscript/` — история редакторских проходов.

**Важно:** папка `docs/active/` упразднена. Её функции теперь распределены между `docs/bible/`, `docs/production/`, `docs/state/`, `docs/filters/`, `docs/story/`. Старые пути `docs/active/*.md` больше не работают — все ссылки в README.md корня указывают на новые места.

По умолчанию перед новой сценой читается минимальный набор, описанный в `../state/project-state.md` и `../production/writing-protocol.md`. Справочные файлы не должны автоматически переопределять последние прямые решения автора.
