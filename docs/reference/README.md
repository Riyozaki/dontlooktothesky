# Справочный архив проекта

Эта папка обозначает материалы, которые не нужно читать перед каждой новой сценой.

Полные канонические рукописи находятся в `docs/manuscript/`. Они открываются адресно для читательского прохода, выплаты факта или проверки непрерывности.

Исторические и подробные документы в корне `docs/` остаются справочниками:

- `SKILL.md` — полная версия голосового контракта;
- `anti-cliche-filter.md` — расширенный анти-ИИ-фильтр;
- `examples.md` — примеры авторского голоса;
- `docs/character-bible.md` — полная библия персонажей;
- `docs/common-route-outline.md` и `docs/story-structure.md` — подробная архитектура;
- `docs/production-method.md` — производственная методика;
- аудиты в `docs/manuscript/` — история редакторских проходов.

По умолчанию читается только `docs/active/`. Справочные файлы не должны автоматически переопределять последние прямые решения автора.
